This curriculum is structured to progress linearly from foundational data concepts to advanced generative AI. Courses are grouped logically by subject matter and prerequisite necessity. Your progress is automatically saved to your browser.

Phase 1: Data Analytics Foundations

This phase establishes basic data literacy, spreadsheet fundamentals, and introductory SQL/visualization before writing complex code.

Phase 2: Core Programming & Version Control

This phase introduces the primary languages (Python and R) and fundamental tools like Git. Note that R programming is front-loaded here to align with the JHU Data Science track.

Phase 3: Applied Python for Data Science & Statistics

Transitioning fully to Python for data manipulation (pandas, matplotlib) and applying inferential statistics/regression.

Phase 4: Introductory Mathematics for Data Science

Establishing the continuous mathematics required for algorithms: essential calculus, foundational linear algebra, and probability distributions.

Phase 5: Introductory Machine Learning & Data Mining

Core classical machine learning building blocks (Supervised/Unsupervised) and basic pattern discovery.

Phase 6: NLP, Text Mining & Social Media Analytics

Specialized coursework dealing with unstructured text, networks, and search retrieval, leveraging the foundational math and ML learned in prior phases.

Phase 7: Data Engineering, Systems & Architecture

Building the infrastructure knowledge (SQL, Linux, Pipelines, Warehousing) required to deploy and manage data at scale.

Phase 8: Advanced Mathematics for AI & Computer Science

Preparation for the CU Boulder MS-AI prerequisite pathway, covering discrete math, advanced multivariable calculus, and principal component analysis.

Phase 9: Practical & Advanced Machine Learning

Deploying complex machine learning models into data products and completing applied machine learning capstones.

Phase 10: Deep Learning, AI Engineering & Generative AI

Advanced neural networks, transformers, reinforcement learning, LLM fine-tuning, and application architecture.
